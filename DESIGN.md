# Chronic Health Manager Design System

## Color Palette

The color palette is rooted in nature-inspired tones to evoke stability and growth.

| Token | Value | Description |
| :--- | :--- | :--- |
| **Primary** | `#0f4361` | Deep Slate Blue: Used for critical actions and primary navigation. |
| **Secondary** | `#246a51` | Sage Green: Applied to success states and health progress indicators. |
| **Tertiary** | `#364245` | Cloud Blue: Used for background washes and subtle grouping. |
| **Background** | `#f8f9fa` | Neutral Grey: Base background color. |
| **Surface** | `#ffffff` | Pristine White: Used for cards and primary containers. |
| **Error** | `#ba1a1a` | Used for error states and critical alerts. |

## Design Philosophy

- **Soft Minimalist:** Reducing cognitive load and medical anxiety through a "Soft Minimalist" approach.
- **Fluid Grid:** 4-column for mobile, 12-column for desktop.
- **Ambient Shadows:** Shadows are tinted with the primary color at low opacity (3-5%) for a natural floating effect.
- **Shapes:** Rounded corners (`0.5rem` base) to remove "sharpness" from medical interfaces. Large containers use `1.5rem` (rounded-xl).
